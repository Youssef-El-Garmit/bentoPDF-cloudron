## About

BentoPDF is a privacy-first, locally hosted PDF toolkit that runs entirely in your browser. All processing happens client-side, ensuring your documents never leave your device. Built with modern web technologies, BentoPDF provides a comprehensive set of PDF manipulation tools without requiring any server-side processing.

## Features

* **Merge PDFs** - Combine multiple PDF files into one
* **Split PDFs** - Split PDFs by page ranges or extract individual pages
* **Rotate PDFs** - Rotate pages in 90-degree increments
* **Compress PDFs** - Reduce file size while maintaining quality
* **Convert Images to PDF** - Convert JPG, PNG, and other images to PDF
* **PDF to Images** - Extract pages as images
* **Add Watermarks** - Add text or image watermarks to PDFs
* **Add Page Numbers** - Add page numbers to your PDFs
* **Remove Pages** - Delete specific pages from PDFs
* **Organize Pages** - Reorder and reorganize PDF pages
* **Add Signatures** - Draw or upload signatures to PDFs
* **Fill PDF Forms** - Fill out PDF forms
* **Extract Text** - Extract text content from PDFs
* **PDF Metadata** - View and edit PDF metadata
* **Password Protection** - Add or remove PDF passwords
* **Privacy First** - All processing happens in your browser, no data is sent to servers
* **No Account Required** - Use immediately without registration
* **Open Source** - AGPL-3.0 licensed, fully open source
